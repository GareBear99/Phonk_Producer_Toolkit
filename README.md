
# Drift Phonk Producer Toolkit

A complete starter toolkit for producing **Drift Phonk / Phonk House**

This repository contains **samples, MIDI patterns, textures, and templates** designed to help producers quickly create phonk-style tracks.

---

# Features

• Key-labeled 808 bass sounds  
• Distorted phonk cowbells  
• House + phonk drum kits  
• Background texture layers  
• MIDI melody patterns  
• Production template  

Everything is organized so you can drop the folders directly into your DAW.

# Production Template

Recommended project settings:

```
BPM: 132
Key: F minor
```

Track layout:

```

Clap/Snare
Hi-Hats

Cowbell Lead
FX
Texture
```

# Basic Song Structure

Typical phonk-house structure:

```
Intro
Drums
Bass
Cowbell Lead
Drop
Variation
Drop
Outro
```

---

# Master Chain (Recommended)

```
Saturation
→ Soft Clipper
→ Limiter
```

This creates the loud **phonk distortion sound**.
